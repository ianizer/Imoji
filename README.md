# Imoji
Locally run Python app for easily saving and copying images to clipboard.

### Why?
For use with Discord. Perhaps there are emojis from another server you want to send but don't have Nitro. If you have media permissions in said server, then you can use Imoji! To use Imoji, you add an image file to it once (or set a directory to check for images), click the image previews that appear in Imoji to copy the images, and voilà! You may now send the copied image to the server as a substitute for an emoji. Save your $3/month!
